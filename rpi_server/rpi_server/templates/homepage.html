Welcome to the RPi TV Noticeboard project

This website serves as a portal for the an RPi TV Noticeboard to fetch images from.

An RPi TV Noticeboard is a simple setup consisting of an RPi connected to a TV with HDMI. The RPi runs a script that periodically fetches images from the server.

To have a look at the genre-wise images uploaded currently, go here.

The code for the project can be found here on Github.
